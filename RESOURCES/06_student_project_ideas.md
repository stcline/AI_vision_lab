# Student Project Ideas

This file collects project ideas that high school students can build using a Raspberry Pi and camera-based AI tools.

## Beginner Projects

- Motion-activated light or alert.
- Object counter for classroom supplies.
- Simple face-detection demo.
- Camera monitor that shows when someone enters a space.
- Pose-tracking demo for movement or exercise.

## Intermediate Projects

- Classroom door visitor counter.
- Desk-area motion detector with logging.
- Gesture-controlled presentation or media tool.
- Object-following camera project.
- Posture or movement coach using pose estimation.

## Advanced Projects

- Sorting system based on detected object type.
- Attendance-style demo with privacy safeguards and approved use.
- Real-time tracking dashboard.
- Multi-class object detection with alerts.
- Physical automation project that reacts to detected motion or pose.

## Good Project Design Rules

- Start with a simple visible outcome.
- Use a project that can be demonstrated in under one minute.
- Keep code modular and well commented.
- Test one feature at a time.
- Build in stages: detect, then track, then react.

## Project Planning Questions

- What should the camera detect?
- What action should happen after detection?
- What data will be saved, if any?
- How will the project be evaluated?
- What part can students customize?

## Classroom Ideas

- Add reflection questions after each build.
- Let student teams present their results.
- Ask students to compare accuracy, speed, and privacy tradeoffs.
- Include a rubric for technical function and explanation.
