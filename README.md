# AI Vision Lab :robot:

A teacher-friendly resource for introducing AI vision systems into secondary classrooms using Raspberry Pi.

This repository is designed for both beginners and experienced instructors. It includes background information, hardware/setup guidance, Chromebook SSH access, AI vision concept overviews, and project ideas for student use.

## What This Repo Covers :bulb:
- Raspberry Pi background and classroom use
- Required and optional equipment
- Setup and installation guidance
- Raspberry Pi package management with `apt` and `pip`
- Chromebook SSH workflows when peripherals are limited
- Core AI vision ideas such as object detection, tracking, motion sensing, face recognition, and pose recognition
- Project ideas suitable for high school students

## Who This Is For :teacher:
- Teachers new to Raspberry Pi or AI vision
- Teachers who want ready-to-use classroom ideas
- Instructors looking for project-based learning with real hardware
